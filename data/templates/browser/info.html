Contributing

Can I contribute Toybox content?

Do you have any old installs of Garry's Mod lying around? If it's from 2010 to 2012, it's possible it has valuable cached downloads we can use to revive more of Toybox. Add Pancakes on {{if .InGame}}Steam{{else}}Steam{{end}} or Discord (@keroronpa), you can help make Cloudbox even better.

What about something else?

At the moment we're only focused on recovering original Toybox uploads.

Garry's Mod 13

How to use

Subscribe to the {{if not .InGame}}Cloudbox addon{{else}}Cloudbox addon{{end}} on the Garry's Mod Workshop. {{if .InGame}}If you're seeing this in-game, you've already subscribed!{{end}}

Compatibility and Bugs

Almost all of the content on Cloudbox is from 2010-2012, scripts may not fully work because of updates. We're actively developing a compatibility layer and releasing improvements often. See the News page for updates.

Commands

You can easily tweak Cloudbox's behavior using {{if .InGame}}Options{{else}}in-game Options{{end}} or console commands.
cloudbox_userchangelevel - Allow non-admins to changelevel to Cloudbox maps
cloudbox_adminonly - Only allow admins to download from Cloudbox
cloudbox_showjoinmessage - Show Cloudbox chat message on join
cloudbox_getpackage - Download a package from Cloudbox by ID and (optionally) revision
cloudbox_purgecache - Purge the Cloudbox downloads cache
