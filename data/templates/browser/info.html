Contributing

Can I contribute Toybox uploads?

Have any old installs of Garry's Mod lying around? If it's from 2010 to 2012, there's a good chance it has valuable cached Toybox creations we can use to revive more of Toybox. Add Pancakes on {{if and (.InGame) (not .GM13)}}Steam (/id/keroronpa){{end}}{{if .GM13}}Steam{{end}}{{if not .InGame}}Steam{{end}} or Discord (@keroronpa), you can help us make our archive even better.

What about something else?

At the moment we're focused on recovering Toybox uploads. If you're playing on Garry's Mod 12, you can upload saves just like you could on Toybox.

Garry's Mod 13

How to use

Subscribe to the {{if not .InGame}}Cloudbox addon{{else}}Cloudbox addon{{end}} on the Garry's Mod Workshop. {{if .GM13}}If you're seeing this in-game, you've already subscribed!{{end}}

Compatibility and Bugs

Almost all of the content on Cloudbox is from 2010-2012, scripts may not fully work because of updates. We're actively developing a compatibility layer and releasing improvements often. See the News page for updates.

Commands

You can easily tweak Cloudbox's behavior using {{if .GM13}}Options (user/server){{else}}in-game Options{{end}}, or console commands.
cloudbox_userchangelevel - Allow non-admins to changelevel to Cloudbox maps
cloudbox_adminonly - Only allow admins to download from Cloudbox
cloudbox_showjoinmessage - Show Cloudbox chat message on join
cloudbox_getpackage - Download a package from Cloudbox by ID and (optionally) revision
cloudbox_purgecache - Purge the Cloudbox downloads cache

Garry's Mod 12

How to use

If you still have a copy of Garry's Mod 12, you can use Cloudbox in-place of the discontinued Toybox. You'll need to edit your HOSTS file (C:\Windows\System32\Drivers\etc\hosts) as admin and add these lines to the end of it:
51.81.34.116 api.garrysmod.com
51.81.34.116 stats.garrysmod.com
51.81.34.116 toybox.garrysmod.com
51.81.34.116 toyboxapi.garrysmod.com
